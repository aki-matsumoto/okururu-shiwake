# Changelog

## [0.6.6] 2026-06-17
### Added
- **HAKU をデパコスマークに追加（v1.2でH欄〇）。** これまで『ハク』が他語と誤一致するため未登録だったが、patterns を『HAKU』英字のみにし、HAKUHODO/白鳳堂/BIHAKU を anti で除外して安全に登録。これで仕分けのデパコスマークが v1.2 の「Recoreまとめ出品〇」と完全一致（THE BODY SHOP含む・HAKU含む）。

## [0.6.5] 2026-06-17
### Fixed
- **H欄〇でない「まとめ出品」誤登録を削除（v1.2定義表で照合）。** LUSH／FANCL・無印良品／Tatcha・Drunk Elephant・The Ordinary は H欄〇でないためデパコスマーク対象から削除。THE BODY SHOP は H欄〇（対象外区分だがまとめ出品〇）なので維持。これで仕分けのデパコスマークが v1.2 の「Recoreまとめ出品〇」と一致（HAKUは『ハク』の誤一致回避で技術的に除外）。

## [0.6.4] 2026-06-17
### Fixed
- **デパコスマークの量販ブランド誤判定を修正（査定拡張と分類統一）。** H欄〇でない量販ブランドが広い親会社名パターン（資生堂/カネボウ）経由でデパコス判定されていたのを除外：ウーノ/アクアレーベル/専科/dプログラム/IHADA/インテグレート/マジョリカマジョルカ/プリオール/ドゥーエ/ツバキ/アネッサ/メディア/フリープラス/スイサイ/アリィー/エビータ。あわせて誤登録の KATE/ヴィセ/セザンヌ・キュレル/dプログラム/IHADA エントリ（H欄〇でない）を削除。本物デパコス（マキアージュ/エリクシール/プレステージ資生堂等）は維持。
- カテゴリ除外regexの「小物」→「服飾小物」に修正（メイク小物カテゴリのデパコスが誤除外されないように。査定F73と同修正）。

## [0.6.3] 2026-06-17
### Changed
- デパコスマークをTier別に表示：Tier1（ハイデパコス10ブランド：シャネル/ディオール/SK-II/クレドポー/トムフォード/エルメスB/ゲラン/YSL/ジバンシイ/ブルガリ）は「💎 ハイデパコス」、その他は「🏬 デパコス」。

## [0.6.2] 2026-06-17
### Fixed
- ITコード印刷を修正：行の「ラベル印刷」ドロップダウン（存在しない/別物）方式を廃止し、ReCORE標準の page-level「在庫ラベルを印刷する」を実行する方式に変更。ITコード=在庫ラベル、PDコード=カタログで別物。一括/個別ボタンとも在庫ラベル印刷を呼ぶ（ケース単位）。

## [0.6.1] 2026-06-17
### Fixed
- デパコスマーク：ファッション雑貨カテゴリ（カテゴリ未取得/デフォルト時含む）では表示しないよう確定ガードを追加。

## [0.6.0] 2026-06-15
### Added
- デパコス対象に「対象外」区分のH欄〇プチプラ系5ブランドを追加（LUSH／FANCL・無印／キュレル・dプログラム・IHADA／KATE・Visee・CEZANNE／Tatcha・Drunk Elephant・The Ordinary）。計46ブランド。
- 誤一致除外(anti)機構：LUSH→BLUSH等、KATE→SKATE/KATE SPADE を除外。広いカナ(ラッシュ)は不採用。

## [0.5.0] 2026-06-15
### Added
- デパコス判定に BVLGARI（ブルガリ・Tier1）を追加。
- マキアージュを独立エントリ（Tier2）に整理（従来は資生堂DSグループ内）。デパコス対象計41ブランド。

## [0.4.2] 2026-06-15
### Changed
- 「まとめ待ち」マークの色をティール→バイオレットに変更（リコア(緑)との視認差を強化）。

## [0.4.1] 2026-06-15
### Added
- まとめ待ちマーク（s8）：未振り分け＋販売1500〜3999円＋未開封/未使用/開封未使用/中古A。
### Changed
- WGA条件を拡張：未振り分け＋（販売¥1/未入力 もしくは 販売1500〜3999円かつ中古B・C）。買取不可は対象外。価格帯ルールは卸/リコア/デパコス無しの商品のみ。

## [0.4.0] 2026-06-15
### Fixed
- デパコス誤判定修正：'HAKU'/'ハク' が「BIHAKU」「ビハク（美白）」に部分一致し、美白系商品に誤ってデパコスマークが付く不具合を修正（マキアージュ／エリクシール等の安全パターンのみに）。
### Added
- WGAマーク（s8）：卸/リコア/デパコスのマークが無く、販売単価が¥1または未入力の商品に「WGA」マークを表示。

## [0.3.1] 2026-06-15

## [0.3.0] 2026-06-15
